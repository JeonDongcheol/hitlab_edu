#### Regression이란?
Regression을 알아보기 앞서 Supervised learning에 대한 개념을 간단하게 살펴본다.

Machine Learning은 프로그램을 학습시키는 분야이다. 프로그램을 학습시키는 방법은 Supervised Learing과 Unsupervised Learning이 있다. Supervised learing은 학습시키고자 하는 label이 있는 예시(데이터)를 제공해 학습시키는 방법이다.  여기서 label이 있는 예시를 training set 이라 부른다.
